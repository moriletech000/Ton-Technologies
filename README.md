# Ton Technology

A responsive educational website for Ton Technologies, an online learning platform focused on computer studies and software development.

## Overview

Ton Tech is a frontend website that showcases courses, services, and testimonials for a computer training institute based in Lagos, Nigeria. The platform offers courses in web design, Microsoft Office tools, and software development.

## Features

- Responsive design across all screen sizes
- Animated navigation with mobile hamburger menu
- Course search with real-time filtering
- Owl Carousel sliders for partners and testimonials
- Newsletter subscription form
- Downloadable course price list
- WhatsApp integration for getting started

## Tech Stack

- HTML5
- CSS3
- JavaScript
- jQuery
- Owl Carousel 2
- Font Awesome 6

## Getting Started

Clone the repository and open `index.html` in any modern browser. No build tools or dependencies required.

```bash
git clone https://github.com/your-username/ton-tech.git
cd ton-tech
```

Then open `index.html` directly in your browser.

## Deployment

This project includes a `vercel.json` configuration file and can be deployed instantly on Vercel.
